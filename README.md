A complete **Exploratory Data Analysis** on the classic Iris Flower Dataset. This project was done as part of my **Data Science Internship**.

---

## 📋 Project Objective

Perform data cleaning, statistical analysis, and visualizations to understand the patterns and relationships between different flower features and species.

---

## 🗂 Dataset Information

- **File Name**: `Iris.csv`
- **Total Rows**: 150
- **Total Columns**: 6
- **Species**: 3 (Iris-setosa, Iris-versicolor, Iris-virginica)
- **Balanced Dataset**: 50 samples per species

### Features:
- `Id`
- `SepalLengthCm`
- `SepalWidthCm`
- `PetalLengthCm`
- `PetalWidthCm`
- `Species`

---

## 🛠 Technologies & Libraries Used

- **Python**
- **Pandas** - Data Manipulation & Cleaning
- **NumPy** - Numerical Operations
- **Matplotlib & Seaborn** - Data Visualization
- **Plotly** - Interactive Plots

---

## 📊 Project Workflow

1. **Data Loading**
2. **Data Cleaning** (Removed `Id` column, checked missing & duplicate values)
3. **Exploratory Data Analysis**
4. **Statistical Analysis**
5. **Data Visualization**

---

## 📈 Key Insights

- **Iris-setosa** is easily distinguishable from the other two species.
- Strong positive correlation between **Petal Length** and **Petal Width**.
- **Iris-virginica** has the largest petal size on average.
- No missing values in the dataset.
- Very few outliers present.
- Balanced classes → Good for classification tasks.
